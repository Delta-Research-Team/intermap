# InterMap

> Accelerated Detection of Interaction Fingerprints on Large-Scale Molecular Ensembles


Hey there! This is the official repository for the InterMap project, which aims to provide an efficient and scalable
framework for detecting interaction fingerprints in large-scale molecular ensembles.

- interaction Fingerprints
- Large-Scale Molecular Ensembles
- Accelerated Detection

## Documentation

All documentation, including installation instructions, usage guides, and visualization options of the results, can be
found at the online [InterMap Documentation](https://Delta-Research-Team.github.io/intermap/).
